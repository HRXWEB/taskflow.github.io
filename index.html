Features

Expressive Programming Models

Taskflow enables developers to express a broad range of computational patterns using minimal programming effort.

Taskflow lets you master most API in a few minutes.

Efficient System Runtimes

Taskflow develops a highly efficient system runtime optimized for latency, energy efficiency, and throughput.

Taskflow lets you focus on high-level development.

Real Applications

Taskflow has demonstrated promising performance in large parallel applications with millions of CPU and GPU tasks.

Taskflow lets you scale to manycore CPUs and GPUs.
